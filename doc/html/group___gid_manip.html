Detailed Description

Group ID manipulators.

GID manipulators are utility functions for inspecting and manipulating group identifiers.

The behavior of GID manipulators depends on the schema version set in the GID. You should make sure the schema version is set before setting other fields in the GID.

To use this module, include the header epid/common/gidmanip.h.
