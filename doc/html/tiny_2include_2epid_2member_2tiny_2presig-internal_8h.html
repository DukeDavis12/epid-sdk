Provides a pointer to precomputed signature in members's pool.

Provides a pre-computed signatures from members's pool if available, otherwise provides a newly calculated pre-computed signature.

Warning
Pre-computed signatures must not be accessed outside of the secure boundary.
You must call MemberPopPreSig() immediately after using the pre- computed signature provided by this function or you risk exposing the member private key.
Parameters
[in]ctxThe member context.
[out]presigThe pre-computed signature removed from members's pool
